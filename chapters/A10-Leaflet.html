Leaflet

0.1 An introduction to using leaflet in R

Leaflet is a popular open-source JavaScript library for interactive maps. It is widely used across the web and is a great way to display location data. In this tutorial, we will learn how to use leaflet in R to create interactive maps.

0.2 The data sets

We will use Department for education progress 8 data to examine school performance in England. We will use two dats sets, one containing the location of schools and the other containing the progress 8 scores for each school.

School location data Progress 8 data

Tip

The progress 8 refers to a value added measure used by the Department for Education in the UK to report the performance of secondary schools. It is calculated by comparing the progress of students from the end of primary school to the end of secondary school. Pupils are placed into bands by their performance at tests in English and mathematics during primary school (the KS2 SATS) and their progress is compared to the average progress of pupils with similar results.

When students sits national examinations at age 16 (GCSEs) their results in English and mathematics and six other subjects (sciences, computer science, geography, history, languages and three further subjects). English and mathematics are double weighted in the calculation of Progress 8. For each student an attainment 8 score is calculated, a measure of average attainment across the eight subjects.

Progress 8 is calculated by subtracting the average attainment 8 score of pupils in the school from the average attainment 8 score of pupils with the same prior attainment at the end of primary school. The difference is divided by ten to give a Progress 8 score. A school with a Progress 8 score of 0 is performing in line with the national average. A school with a positive Progress 8 score is performing better than the national average and a school with a negative Progress 8 score is performing worse than the national average.

0.3 Loading the data and merging the data sets

The first step is to load the two data sets. Often in projects data may be spread across multiple sources - here the location data and the progress 8 data are in separate files. We will load the data and then merge the two data sets together.

We don’t need all the columns in the data sets, and will keep the following entries from the two data sets:

# Select variables of interest
# Note the URN in both data sets to perform the merge

school_descriptive <- school_descriptive %>%
  select(year, URN, SCHNAME, Postcode, LA, Easting, Northing, NFTYPE)

progress_8_data <- progress_8_data %>%
  select(URN, P8MEA, PSENE4)

We will filter the school_descriptive data for 2018 data only and for secondary schools which have returned Progress 8 data.

# Filter the descriptive data
school_descriptive <- school_descriptive %>%
  filter(year == 2018)

progress_8_data <- progress_8_data %>%
  filter(!is.na(P8MEA)) %>%
  filter(P8MEA != "NP" & P8MEA != "NE" & P8MEA != "SUPP")

We can then merge the two data sets together using the left_join() function from the dplyr package. The left_join() function is used to combine two data frames by matching the values of one or more variables. In this case, we will merge the two data sets using the URN variable.

# Merge data sets

merged_data <- left_join(progress_8_data, school_descriptive, by = "URN")

0.4 Converting coordinates to latitude and longitude

A common issue with geographical data is that the coordinates are often in a different from those used by mapping software. For example, the coordinates in the data set are in eastings and northings rather than latitude and longitude. We will use the sp package to convert the eastings and northings to latitude and longitude. The sp package is a popular package for working with spatial data in R.

First we use the coordinates() function in R, from by the sp package, which is used to assign or retrieve spatial coordinates for a dataset

# Converting Eastings and northings to latitude and longitude

library(sp)

# Omit any schools with missing eastings or northings

merged_data <- merged_data %>%
  filter(!is.na(Easting) & !is.na(Northing))

# Create a SpatialPoints object from the eastings and northings
coordinates(merged_data) <- c("Easting", "Northing")

# Define the CRS for British National Grid (EPSG:27700)
proj4string(merged_data) <- CRS("+proj=tmerc +lat_0=49 +lon_0=-2 +k=0.9996012717 
                        +x_0=400000 +y_0=-100000 +ellps=airy 
                        +datum=OSGB36 +units=m +no_defs")

# Transform to WGS84 (latitude and longitude)
df_ll <- spTransform(merged_data, CRS("+proj=longlat +datum=WGS84"))

# Add latitude and longitude to the original data frame
merged_data$longitude <- coordinates(df_ll)[, 1]
merged_data$latitude <- coordinates(df_ll)[, 2]

0.5 Creating an interactive map with leaflet

Now that we have the data in the correct format, we can create an interactive map using the leaflet package in R.First, we will create a map that displays the location of schools in England. We will use the leaflet() function to create a new leaflet map object, and then add markers to the map using the addMarkers() function.

First let us just create a basic map around the Waterloo campus of KCL.

library(leaflet)

# Create a blank leaflet map

leaflet() %>%
  addTiles() %>%  # Add the default OpenStreetMap tiles 
  setView(lng = -0.1134284, lat =  51.50556315 , zoom = 16)  # Set the view by adding the lat and long of Waterloo

You can change the style of the map by adding different tile layers. For example, you can use the addProviderTiles() function to add a different tile layer to the map. The addProviderTiles() function allows you to add tiles from different providers. See options for tile styles here. You can change the level of zoom in the setView() function.

# Create a blank leaflet map in a different style

leaflet() %>%
  addProviderTiles(provider = "OpenTopoMap") %>%  # Add OpenTopoMap tiles 
  setView(lng = -0.1134284, lat =  51.50556315 , zoom = 14)

0.6 Adding markers to the map

The simplest way to add data to the map is to use the addCircles() function. This function allows you to add a cricle to the map at specific locations. You can customize the markers by changing the color, size, and shape of the markers. You can also add popups to the markers to display additional information.

# Create a map with a marker

leaflet() %>%
  addTiles() %>%  # Add the default OpenStreetMap tiles 
  setView(lng = -0.1134284, lat =  51.50556315 , zoom = 16) %>%
  addCircles(
    lng = -0.1134284, 
    lat = 51.50556315, 
    radius = 50,  # Radius in meters
    color = "red",  # Circle outline color
    fillColor = "red",  # Circle fill color
    fillOpacity = 0.5  # Opacity of the fill
  )

Alternative, you can use the addMarkers() function to add markers to the map. The addMarkers() function allows you to add markers to the map at specific locations. You can customize the markers by changing the icon, size, and shape of the markers. You can also add popups to the markers to display additional information.

# Create a leaflet map wtih a marker with a popup label

leaflet() %>%
  addTiles() %>%  # Add the default OpenStreetMap tiles 
  setView(lng = -0.1134284, lat =  51.50556315 , zoom = 16) %>%
  addMarkers(
    lng = -0.1134284, 
    lat = 51.50556315, 
    popup = "This is the Waterloo Bridge Wing"  # Optional popup text
  )

0.7 Adding data from a data frame to maps

Rather than simply adding markers, we can add data from a data frame to the map. We can use the addMarkers() function to add data from our school data frame to the map. This time we will pipe (%>%) the merged_data data frame into leaflet. We will use the addMarkers() function to add markers to the map at the location of each school. We will also add a popup to each marker that displays the name of the school.

# Create a map with data from a data frame

merged_data %>%
  leaflet() %>%
  addTiles() %>%
  setView(lng = -0.1134284, lat = 51.50556315, zoom = 13) %>%
  addMarkers(
    lng = ~longitude,   # Use the 'longitude' column from your data frame
    lat = ~latitude,    # Use the 'latitude' column from your data frame
    popup = ~as.character(SCHNAME)    # Use the 'SCHNAME' column for the popup
  )

Alternatively, you can use the addCircleMarkers() function to add markers to the map. The addCircleMarkers() function allows you to add circle markers to the map at specific locations. You can customize the markers by changing the color, size, and shape of the markers. You can also add popups to the markers to display additional information. Here we set the fill to the type of school.

We need to specify a palette for the fill. We can use the colorFactor() function from the leaflet package to create a color palette based on the NFTYPE variable in the data frame. The colorFactor() function creates a color palette based on the unique values in a variable. We can then use the fillColor argument in the addCircleMarkers() function to set the fill color based on the NFTYPE variable.

The NFTYPE variable can have the following values:

# set a pallete for the fill

# Define a custom color palette (use your preferred colors)
custom_colors <- c("#FF5733", "#33FF57", "#3357FF", "#F1C40F", "#9B59B6", "#1ABC9C", "#E74C3C", 
                   "#F39C12", "#D35400", "#8E44AD", "#3498DB", "#2ECC71", "#16A085", "#9B59B6",

pal <- colorFactor(
  palette = custom_colors,  # Set the colors for the palette
  domain = merged_data$NFTYPE  # Set the domain for the palette
)

# Create a map with data from a data frame

merged_data %>%
  leaflet() %>%
  addTiles() %>%
  setView(lng = -0.1134284, lat = 51.50556315, zoom = 13) %>%
  addCircleMarkers(
    lng = ~longitude,   
    lat = ~latitude,   
    radius = 8,         
    color = NA, # No outline coulor      
    fillColor =  ~pal(NFTYPE),  # Use the palette to color the markers by NFTYP
    fillOpacity = 0.8,  
    popup = ~ as.character(SCHNAME)  # Use the 'SCHNAME' column for the popup    
  )

We can follow a similar process to add the progess 8 data to the map. This time we will create a continuous colour scale to show variation in progress 8 score. First let us convert P8MEA to numeric and check its range:

# Convert P8MEA to numeric

merged_data$P8MEA <- as.numeric(merged_data$P8MEA)

# Check the range of P8MEA
range(merged_data$P8MEA, na.rm = T)
[1] -3.84  2.55

We can now define a continuous colour palette using the colorNumeric() function from the leaflet package. The colorNumeric() function creates a continuous color palette based on a numeric variable. We can then use the fillColor argument in the addCircleMarkers() function to set the fill color based on the P8MEA variable. There are a number of options for the palette, in the colorNumeric() function see here for a full list. Palettes include:

  • “YlGnBu”: Yellow to green to blue
  • “RdYlBu”: Red to yellow to blue (good for diverging data)
  • “Viridis”: A perceptually uniform color map (great for continuous data)
  • “Magma”: A perceptually uniform palette with darker tones
  • “Plasma”: Bright and perceptually uniform
  • “Inferno”: A bright, perceptually uniform palette
# Create a continuous color palette based on P8MEA values
pal <- colorNumeric(
  palette = "RdYlBu",  # Choose the palette (can be changed to any desired palette)
  domain = c(-3.84, 2.55)  # Set the domain to the range of P8MEA
)

# Create the leaflet map
merged_data %>%
  na.omit() %>% # filter NAs
  leaflet() %>%
  addTiles() %>%
  setView(lng = -0.1134284, lat = 51.50556315, zoom = 13) %>%
  addCircleMarkers(
    lng = ~longitude,    # Use the 'longitude' column from your data
    lat = ~latitude,     # Use the 'latitude' column from your data
    radius = 10,          # Radius of the circle markers
    color = NA,          # Remove the outline by setting color to NA
    fillColor = ~pal(P8MEA),  # Use the continuous color scale for fillColor
    fillOpacity = 0.9,   # Opacity of the fill
    popup = ~as.character(SCHNAME)     # Show the school name in the popup
  ) %>%
  addLegend(
    position = "bottomright",   # Position of the legend
    pal = pal,                  # The color palette
    values = merged_data$P8MEA, # The values to use in the legend
    title = "P8MEA",            # Title of the legend
    opacity = 1                 # Opacity of the legend
  )
